# REPORT — Blog News Microservices

## Описание проекта

Микросервисное приложение для управления новостным Telegram-каналом.
Редактор публикует посты через Telegram-бота, система обрабатывает контент и уведомляет подписчиков по email.

---

## Архитектура

| Сервис | Технология | Назначение |
|:---|:---|:---|
| `telegram-bot` | Python (aiogram) | Интерфейс редактора, публикация в канал |
| `media-service` | Go + Fiber | CDN-хранилище медиафайлов с метаданными |
| `post-service` | Go + PostgreSQL | Хранение текстов постов |
| `user-service` | Go + PostgreSQL | Регистрация и хранение email-подписчиков |
| `notification-service` | Go + SMTP | Отправка email-уведомлений, обратная связь редактору |
| `automation-service` | Go + RabbitMQ + Redis | Брокер сообщений + кэш подписчиков (сервис автоматизации) |
